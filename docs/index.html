Overview

Harbour is the open/free software implementation of a multi-platform, multi-threading, object-oriented, scriptable programming language, backwards compatible with xBase languages. Harbour consists of a compiler and runtime libraries with multiple UI and database backends, its own make system and a large collection of libraries and interfaces to popular APIs. {% unless site.fork %}

Harbour is a cross-platform compiler and is known to compile and run on Windows (32/64-bit), GNU/Linux, macOS, Android, OS/2 and MS-DOS. {% endunless %} Read more ⇢
